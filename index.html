SPUG Southampton Python Users Group

Southampton Python Users Group (SPUG) is a general users group for all things Python. It is based primarily at the University of Southampton, but is open to anyone. We usually have an invited guest speaker once a month to lead a session. We also get together informally in small groups to work on collaborative projects, or to help each other out. We communicate mainly through Slack, but have a mailing list for the important stuff. We also have a GitHub group for sharing code. SPUG is open to absolutely anyone who is interested in Python regardless of experience or level.

Get involved

Google Group
Anyone can join the Google group. This serves as our main mailing list and will be the main channel of communication for upcoming events. Simply follow the link. Ideally you'll need a Google account to join, although it is possible without. Note that it is not necessary to have a Gmail account to sign up to Google.

Google Group »

Slack Channel
Slack is a messaging app for groups. We use it for less important communications, such as asking other group memebers for help, or for arranging small, less formal meetups, or sharing our work and knowledge with others. If you have an email address ending in soton.ac.uk you can sign yourself up, otherwise you need to be invited. We're happy to do this for you, simply join the Google group above and send Simon Tudge a quick email.

Slack Group »

Github page
GitHub is a website for hosting and sharing code, and is built on the source control software git. SPUG has its own GitHub group where we often share code snippets, or speakers may post the material for their talks. To join you need to create a GitHub account, which is easy and completely free, and then send us your username so we can add you to the group. Note that you can download all the code without actually signing up.
